What are difference among map(), filter(), find() and forEach()?

Ans: map,filter,find and forEach are array methods.They all are higher order method; But they are very different in the point of their actions.Like, map() returns array, filter() also returns array based on condition,find() returns single element based on condition and forEach() returns undefined.

What are difference among localStorage, sessionStorage and cookies?

Ans: localStorage, sessionStorage and cookies are web storage where can store data or information.But they work differently.We can store information in local storage for long time, sessionStorage can store data for short time.

What is Event loop and how does it work?

Ans: Event loop is very important in javaScript.There are many event occurs in a web app.It can catch by set handler.But all handler do not execute same time.They execute one by one means it runs or loop until all handler completed.
